Neural-Net Operators
Summary ›

The core operators behind CNNs and transformers — each with an animation of how it works and an efficient, branch-free C++ kernel. The code style reflects Quadric's Chimera GPNPU: one unified programmable core runs matrix, vector, and scalar work in a single instruction stream across a wide PE grid, so operators are written as fused data-parallel loops with no data-dependent branches (activations use max/min/select; boundaries use lane-position masks), and quantized int8×int8→int32 MACs with a requantize step. Every C++ snippet was compiled and run before it was pasted here.

Interview prep · animations are canvas-drawn, dependency-free · C++ verified with g++ 13.
